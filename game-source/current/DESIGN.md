# 钓鱼插件图片渲染设计文档

## 图片渲染尺寸规范

### 核心发现

**html_to_pic 渲染引擎使用 devicePixelRatio = 2 进行渲染**

所有通过 `nonebot_plugin_htmlrender` 的 `html_to_pic` 渲染的图片，实际输出尺寸是代码设置尺寸的 **2倍**。

### 尺寸对照表

| 项目 | 代码设置 (CSS) | 实际输出 |
|------|---------------|---------|
| 场景宽度 | 330px | 660px |
| 场景高度 | 185px | 370px |
| 人物宽度 | 22px | 44px |
| 人物高度 | 61px | 122px |

### 解决方案：50%缩小无损渲染

**关键结论：将图片在CSS中缩小50%显示，经过2倍放大后可完美还原原始像素。**

#### 验证结果

| 原始图片尺寸 | CSS显示尺寸 | 实际输出尺寸 | 效果 |
|-------------|------------|-------------|------|
| 30×55px | 30×55px (100%) | 60×110px | 像素放大，模糊 |
| 30×55px | 15×27.5px (50%) | 30×55px | **完美还原，无损** |

#### 实现方式

```css
/* 原始图片 30×55px，缩小50%显示 */
.sprite-half {
    width: 15px;
    height: 27.5px;
    image-rendering: pixelated;
}
```

```html
<img src="sprite.png" class="sprite-half">
```

### 设计原则

1. **素材准备**：所有像素风素材应按 **2倍尺寸** 准备
2. **CSS设置**：在CSS中将尺寸设置为实际尺寸的 **50%**
3. **渲染输出**：html_to_pic 自动放大2倍，还原为原始像素尺寸

### 当前代码中的尺寸设置

位置：`utils.py` 第1208-1210行

```python
container_width = 330  # 场景容器宽度，输出660px
sprite_w = 22          # 人物宽度，输出44px
sprite_h = 61          # 人物高度，输出122px
```

### 新素材适配

如需使用新的高清素材（如 `frame_XXX.png` 系列，30×55px）：

```python
# 新素材原始尺寸：30×55px
# CSS中设置50%：15×27.5px
# 输出尺寸：30×55px（完美还原）

sprite_w = 15    # 30 / 2 = 15
sprite_h = 27.5  # 55 / 2 = 27.5
```
