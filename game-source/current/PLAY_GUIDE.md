# 钓鱼插件游玩指南

## 基础指令

### 钓鱼相关
| 指令 | 格式 | 说明 |
|------|------|------|
| 开始钓鱼 | `钓鱼 [地点编号]` | 开始钓鱼，不填编号会显示地点选择 |
| 收杆 | `收杆` | 结束钓鱼并结算收益，结算时扣除本次消耗的鱼饵；不足一条鱼上钩的时间按动态概率判定 |
| 钓鱼状态 | `钓鱼状态` | 查看当前钓鱼状态和收益；每次查看都会结算、扣除鱼饵并重置计时器 |
| 钓鱼签到 | `钓鱼签到` | 每日签到获取玉米 |

> 注意：`钓鱼状态` 和 `收杆` 都会产生结算，并扣除本次消耗的鱼饵。不足一条鱼上钩的时间会以动态概率判定是否上鱼，结算后重置钓鱼计时器；持续观测状态可能导致长时间不上钩。

## 游戏技巧

1. **新手入门**：从「乡间浅溪」开始，积累钓鱼币升级装备
2. **装备优先**：优先升级钓竿提升稀有鱼概率，再升级鱼钩提升速度
3. **鱼饵选择**：根据钓鱼时长选择合适的鱼饵，长时间钓鱼用高级鱼饵更划算
4. **打窝技巧**：打窝可提升稀有鱼概率，适合在高级钓场使用
5. **展示收益**：将稀有鱼放入展示区可获得每日被动收益
6. **每日签到**：记得每日签到获取玉米，玉米可用于打窝
