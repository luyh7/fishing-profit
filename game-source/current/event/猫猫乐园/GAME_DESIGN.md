# 猫猫乐园 · 游戏设计规范文档

> 版本: v4 / 最后更新: 2026-05-25 / 状态: 设计完成, 待实现

---

## 一、概述

### 1.1 活动定位

猫猫乐园是钓鱼游戏的**永久常驻中型活动**。玩家在收集田园池塘全部 UTR 后获得门票，进入乐园参与建设。9 栋功能各异的建筑逐步从空地变为游乐园，建成后转换为每日签到收益的运营模式。

### 1.2 核心设计理念

- **唯一重奖**: 鱼竿等级 +1（活动外生效，记为额外加成不影响商店升级价格），作为活动的最終驱动
- **低保底收入**: 材料大量掉落污染背包，鱼价压低至正常 70%，玩家在乐园中的净收益有限
- **图鉴驱动力**: 10 种鱼 × 6 稀有度 = 60 格图鉴，收集耗时远超建设周期，防止钓鱼过程单调
- **建筑即玩法**: 9 栋建筑不是数值叠加，每栋改变一种游戏机制
- **雕像即冠冕**: "传奇猫雕像" 始终是最后建成的，所有建筑升级受其门控

### 1.3 建设周期目标

| 指标 | 目标值 | 现模拟值 |
|------|--------|---------|
| 平均建设天数 | ~20 天 | 20.4 天 |
| 18~22 天命中率 | ≥90% | 94% |
| 最短/最长 | 17~26 天 | 17~26 天 |
| 竣工时图鉴收集 | — | ~39/60 (65%) |

## 三、核心循环

### 3.1 每日流程

```
┌─ 每日开始 ──────────────────────────────────────────┐
│ 1. 随机天气（6选1）                                   │
│ 2. 猫咪摩天轮签到（0~3抽，取决于建筑等级）               │
│ 3. 计算当日钓鱼参数（汇总所有建筑 + 天气效果）            │
│ 4. 12小时钓鱼模拟 → 鱼获 + 材料                        │
│ 5. 玩家手动选择建筑升级（消耗材料，即时完成）              │
│ 6. 日志展示本日成果                                    │
└─────────────────────────────────────────────────────┘
```

### 3.2 每竿产出的互斥判定

每次收竿只产生**一种**结果：

```
随机数 < 材料掉率  →  获得随机材料 ×1
     ↓ 否
钓到一条鱼（随机鱼种 + 稀有度 + 价格）
```

材料与鱼获互斥。当前落地版本材料率随猫爬架广场等级动态变化：Lv0=30%, Lv1=35%, Lv2=40%, Lv3=45%（猫爬架广场升级后材料率递增，鱼获率相应递减）。

### 3.3 每日钓鱼量计算

```
基础间隔 = 60s / (1 + HOOK×10%) / (1 + 鱼饵加成)
实际间隔 = 基础间隔 / (1 + 总速度加成%)
每日竿数 = 12 × 3600 / 实际间隔（取整）
```

初始参数: rod=7, hook=7, 传说鱼饵(+120%), 基础间隔 16.0s, 日均 ~2,692 竿。

## 四、鱼种与稀有度

### 4.1 10 种猫猫乐园主题鱼

```
橘座鲫鱼 / 暹罗鳊鱼 / 布偶白条 / 奶牛麦穗 / 三花泥鳅
金渐层锦鲤 / 英短青鱼 / 无毛鲶鱼 / 折耳翘嘴 / 卷耳黄颡
```

### 4.2 6 种稀有度 & 稀有度分布

`d = rod_level - 公园难度(6)`, 初始 rod=7 → d=1

| d | N | R | SR | SSR | UR | UTR |
|----|------|------|------|------|------|------|
| 0 | 66.6% | 33.5% | 0% | 0% | 0% | 0% |
| 1 | 49.6% | 42.5% | 7.9% | 0% | 0% | 0% |
| 2 | 31.5% | 50.4% | 18.1% | 0% | 0% | 0% |
| 3 | 16.4% | 48.9% | 34.7% | 0% | 0% | 0% |
| ... | ... | ... | ... | ... | ... | ... |

> 高稀有度在正常钓鱼下几乎不出。SSR/UR 需通过水晶猫城堡竿级+1、流星天气加成、迷途风 UTR 特殊判定来获取。这是设计意图——图鉴收集是一个远比建设更长的长期目标。

### 4.3 鱼价公式（低保底）

```
基础价表: N=8, R=22, SR=55, SSR=120, UR=220, UTR=400
场景系数 = 1 + 6×0.3 = 2.8
鱼种系数 = 1 + rand(0~9)×0.1
价格 = 基础价 × 0.7(乐园折扣) × 2.8(场景) × 鱼种系数

最终均价:
  N≈11, R≈30, SR≈76, SSR≈166, UR≈304, UTR≈552
```

鱼饵每竿 50 金 → 绝大多数鱼净亏。玩家钓的是材料，不是钱。

## 五、6 种天气系统

| 天气 | 出现概率 | 基础效果 | 猫猫过山车增幅后(max) |
|------|---------|----------|----------------------|
| 晴天 | 25% | 无特殊效果 | 无 |
| 雨天 | 20% | 钓鱼速度 +10% | +12% |
| 流星 | 15% | 最高稀有度概率 +2% | +2.4% |
| 暴雨 | 15% | 鱼饵消耗减免 50% | 60% |
| 迷途风 | 10% | 1.0% 概率 UTR | 1.2% |
| 猫！ | 15% | 15% 概率猫吃掉鱼 | 18% |

- 猫猫过山车按 `基础效果 × (1 + 过山车增幅%)` 计算
- 暴雨与猫猫小木屋的饵节省采用独立概率叠加: `1 - (1-木屋率) × (1-暴雨率)`
- 流星效果是"最高非零稀有度 +boost, 次高 -boost"
- 迷途风 UTR 是额外判定：先正常投稀有度，再有一次 UTR 判定（成功则覆盖）

### 7.2 传奇猫雕像门控规则

传奇猫雕像的升级条件依赖于**其他 8 栋建筑的总体进度**：

| 雕像升级 | 需要其他8栋 | 解锁效果 |
|---------|------------|---------|
| 雕像 Lv0 → Lv1 | 其他8栋全部 ≥ Lv1 | 所有建筑解锁 Lv2 |
| 雕像 Lv1 → Lv2 | 其他8栋全部 ≥ Lv2 | 所有建筑解锁 Lv3 |
| 雕像 Lv2 → Lv3 | 其他8栋全部 ≥ Lv3 | 鱼竿等级 +1（额外加成，不影响商店升级价格） |

对于普通建筑：
- Lv1: 随时可建，无前置条件
- Lv2: 需要雕像 ≥ Lv1
- Lv3: 需要雕像 ≥ Lv2

**实际建设顺序**: 玩家先建其他8栋 Lv1 → 雕像 Lv1 → 其他8栋 Lv2 → 雕像 Lv2 → 其他8栋 Lv3 → 雕像 Lv3 → 竣工。

| 摩天轮等级 | 每日抽取次数 | 奖池（均匀随机） |
|-----------|------------|-----------------|
| Lv0(未建) | 0 | — |
| Lv1 | 1次 | 真多多药水 / 时光药水 / 幸运药水 / 猫框 / 玉米 |
| Lv2 | 2次 | 同上 |
| Lv3 | 3次 | 同上 |

> 签到奖励对活动外生效。20 天建设期内 Lv1 约 7 天, Lv2 约 10 天, Lv3 约 3 天 → 总计约 7×1 + 10×2 + 3×3 = 36 抽。

## 九、与主世界钓鱼的异同

| 属性 | 主世界 | 猫猫乐园 |
|------|--------|---------|

| 天气 | 共享天气系统 | 共享天气系统 |
| 稀有度 | N~UTR | N~UTR（d=1 基准，SR+靠建筑/天气） |
| 产出 | 纯鱼获 | 鱼获 + 建筑材料（互斥） |
| 材料用途 | — | 建造/升级乐园建筑 |
| 建筑 Buff | — | 仅乐园内生效（除签到、鱼竿+1） |
| 图鉴 | 主世界图鉴 | 乐园独立图鉴（10鱼×6稀有=60格） |

## 十、建设节奏分析

80 次批量模拟结果（0.85% 材料掉率, 12h/天）:

| 阶段 | 典型天数 | 说明 |
|------|---------|------|
| 其他8栋 Lv1 | Day 1~5 | 快速获得基础收益 |
| 雕像 Lv1 | Day 5~6 | 解锁 Lv2 |
| 其他8栋 Lv2 | Day 6~14 | 核心建设期 |
| 雕像 Lv2 | Day 14~15 | 解锁 Lv3 |
| 其他8栋 Lv3 | Day 15~19 | 冲刺期 |
| 雕像 Lv3 → 竣工 | Day 19~20 | 鱼竿+1 |

平均 20.4 天, 94% 置信落在 18~22 天区间。

## 十一、建设顺序策略建议

玩家可以根据自身偏好选择优先升级的建筑：

| 流派 | 优先升 | 原因 |
|------|--------|------|
| 早起鸟 | 猫咪摩天轮 | 每天多 1~3 次签到抽奖 |
| 省钱党 | 猫猫小木屋 | 省鱼饵 = 省金币 |
| 速攻党 | 喵喵鱼塘 | 更快钓竿 → 更多鱼 → 更多钱 |
| 材料党 | 猫爬架广场 | 更高材料掉率 → 更快建 |
| 图鉴党 | 水晶猫城堡 | 竿级+1 → 出 SR/SSR |
| 天气玩家 | 猫猫过山车 | 增强天气效果 |

- 可随时返回主世界继续钓鱼
