| 项 | 方向 |
|----|------|
| 地图 | 21～30，difficulty 20～29，海洋主题 |
| 战斗 | 收杆/状态 **自动结算**，不连点回合 |
| 编队 | 固定三角：炮手 / 护卫 / 军师 各 1 |

| 经济 | 海战币期望显著低于纯钓鱼；失败只动本段鱼获 |
