# S3 · 鼠影海域 — 游戏设计文档

> 版本: **v0.1 策划草案**  
> 状态: 设计讨论稿（先定玩法与边界，后做战斗模拟与落地）  
> 范围: **主线钓场 21～30 图** + **海域专属子系统：三猫护航 / 鼠海盗**  
> 类型: QQ 群低操作挂机钓鱼扩展（战斗自动结算，不走回合连点）  
> 独立性: **猫咪与战斗仅在 21～30 图生效**；1～20 图与 S1 完全不受影响

---

## 0. 一句话

玩家从星海尽头驶入**鼠影海域**继续钓鱼。  
水里仍是鱼，水面上却有**老鼠海盗**登船打劫——  
于是你招募恰好三只各司其职的猫咪，组成护航小队：你负责甩竿，猫负责打鼠。

## 1. 与 S1 / S2 / 主线的定位

| 项目 | S1 猫猫乐园 | S2 星穹矿脉 | **S3 鼠影海域（本季）** |
|------|------------|------------|------------------------|
| 形态 | 常驻活动地图 `S1` | **完全独立** 挂机小游戏 | **主线第三段地图 21～30** |
| 入口 | 特殊地图 | 独立指令 | `钓鱼 21`…`钓鱼 30` |
| 鱼竿 | 需要 | 不需要 | **需要**（钓鱼层） |
| 鱼币 | 流通 | 不关联 | **流通**（钓鱼层 + 海盗战利品） |
| 核心玩法 | 建设乐园 | 指数挖矿破界 | 海域钓鱼 + **三猫自动海战** |
| 作用范围 | 仅 S1 内建筑 buff | 独立表 | **战斗/猫咪仅 21～30** |

### 1.1 设计原则

1. **旧图零侵入**：任何猫咪等级、职业、海战 buff、海盗掉落加成，**不得**改变 1～20 / S1 的速度、鱼价、稀有度、材料。  
2. **QQ 友好**：不要求多轮对话对战；默认在 `钓鱼状态` / `收杆` 时**自动结算**遭遇。  
3. **双层经济**：外层仍是钓鱼期望曲线（延续 `doc/概率和数值设计文档.md`）；内层是「护航战力 → 少被抢、多缴获」。  
4. **职业固定三角**：永远是 3 个槽位、3 种职业各 1，不搞自由 6 人编队。  
5. **先策划后模拟**：本文定规则与接口；数值表由后续 `demo/battle_sim.py` 跑通后再锁。

## 2. 叙事设定

### 2.1 世界

星空第二季尽头的「奇迹彼岸」外，是一片被鼠影旗遮住月光的**开放海域**。  
表面上是渔场，暗处是「吱吱黑旗」船团：它们不钓鱼，只抢别人钓好的鱼和补给。

猫咪们原本在乐园晒太阳，被征召上船后才发现——  
打老鼠居然比逗猫棒还上瘾。

### 2.2 调性

- 轻松、猫、罐头、海风，**轻微海盗喜剧**，不走黑暗写实。  
- 海盗全是**老鼠拟人**（船长鼠、火药鼠、会计鼠……），战败文案搞笑为主。  
- 文案保持与 S1/S2 一致的「猫本位」语气。

## 5. 核心循环（玩家一天在做什么）

```
解锁 21+ → 组好三猫护航队（低频）
        ↓
  钓鱼 2x～3x（挂机，与旧图相同）
        ↓
  状态/收杆时自动结算：
     · 正常出鱼（钓鱼层）
     · 期间遭遇 N 次鼠海盗（海战层）
        ↓
  胜 → 缴获海盗箱/材料/少量币
  负 → 被抢走部分鱼获或饵，文案吐槽
        ↓
  用材料升级三猫 / 换更好的猫
        ↓
  推进图鉴 → 开下一海域图
```

可选主动玩法（不强制）：

| 指令 | 频率建议 | 作用 |
|------|----------|------|
| `海域` / `护航` | 随时 | 面板：三猫、战力、今日遭遇统计 |
| `派猫` / `护航编队` | 低频 | 查看/更换三职业槽位上的猫 |
| `海战` | 每日 0～3 次 | **主动出击**附近海盗船（额外奖励，占日次数） |
| `海域图鉴` | 随时 | 鼠海盗种类 / 海域鱼 |
| `海域帮助` | 随时 | 规则 |

> 主动 `海战` 可砍 v1，只留自动遭遇；建议 v1 就保留，因「点一下打海盗」反馈强且仍低操作。

## 6. 三猫护航系统

### 6.1 槽位与职业（定案方向）

固定 **3 槽 · 三角色**，不可重复职业上阵：

| 槽位 | 职业 | 战斗职责 | 对钓鱼层的间接影响（仅 21～30） |
|------|------|----------|--------------------------------|
| 1 | **炮手猫** | 主输出：提高胜率、提高战利品档 | 胜率↑ → 更少被抢 |
| 2 | **护卫猫** | 主减伤：失败时少丢鱼/饵 | 失败惩罚↓ |
| 3 | **军师猫** | 控场/连胜：提高暴击缴获、降低遭遇骚扰次数或提高「完美护航」率 | 净效率↑ |

> 职业名可皮成：火炮喵 / 水手喵 / 海图喵；内部 id 建议 `gunner` / `guard` / `tactician`。

### 6.2 猫咪实体

每只猫是一条可收集单位（独立于 S1 乐园建筑、也独立于 S2 矿工）：

```text
CatCard {
  id, name, profession,      # 职业固定，抽到即绑定
  rarity,                     # N/R/SR/SSR/UR/UTR 或 1~5 星（实现二选一，建议 1~5 星更轻）
  level,                      # 1..max，用海域材料升级
  base_atk, base_def, base_sup,
  skill_id,                   # 小技能，被动为主（QQ 不点技能）
  skin?                       # 可选外观
}
```

**上阵规则**：

- 每个职业槽只能上 1 只该职业的猫；  
- 未上阵的猫进「船员舱」纯收藏/备选；  
- 更换编队：`派猫 炮手 猫名/编号`（冷却或每日限次，防刷屏可选）。

### 6.3 战力聚合（草案公式，待模拟）

\[
P_{\mathrm{atk}} = \mathrm{Gunner}_{\mathrm{atk}} \cdot (1 + 0.15\cdot L_g)
\]
\[
P_{\mathrm{def}} = \mathrm{Guard}_{\mathrm{def}} \cdot (1 + 0.15\cdot L_d)
\]
\[
P_{\mathrm{sup}} = \mathrm{Tactician}_{\mathrm{sup}} \cdot (1 + 0.15\cdot L_t)
\]
\[
\mathrm{Power} = 0.45 P_{\mathrm{atk}} + 0.35 P_{\mathrm{def}} + 0.20 P_{\mathrm{sup}}
\]

- 缺槽（某职业空）：该维按 0 计，并附加 **组队不完整惩罚**（胜率 ×0.5）。  
- 初始白给三猫保证 Power 略高于 21 图杂兵，略低于 25 图精英。

### 6.4 猫咪获取

| 来源 | 内容 | 备注 |
|------|------|------|
| 首登 21 图 | 三职业各 1 只 N/1 星 | 必给 |
| 自动海战 / 主动海战 | 低概率猫碎片或整卡 | 主循环 |
| 海域材料合成 | 固定配方升星/升稀有 | 保底成长 |
| 图鉴里程碑 | 21/25/30 图收集奖猫 | 中长期 |
| （可选）鱼币+材料限购 | 每周 1 次船员招募 | 防肝 |

**不与 S1 建筑材料互通**（毛线团等不能直接当海战材料），避免污染旧活动。

## 7. 鼠海盗与战斗结算

### 7.1 何时触发（自动）

玩家在 **21～30** 挂机钓鱼时，结算引擎按本次时长计算遭遇次数：

\[
N_{\mathrm{enc}} = \left\lfloor \frac{T_{\mathrm{minutes}}}{T_{\mathrm{base}}(map)} \cdot \mathrm{threat} \cdot \mathrm{noise} \right\rfloor
\]

- \(T_{\mathrm{base}}\)：基准「每多少分钟 1 次遭遇」（21 图可 40～60 分钟，30 图 25～35 分钟，待模拟）。  
- `noise`：0.85～1.15 轻微抖动，避免死板。  
- **军师猫**可降低有效遭遇次数，或把部分遭遇改为「被提前吓跑」（无惩罚无奖励）。  
- 单次收杆遭遇次数设 **软顶**（如 ≤8），防止 10 小时收杆刷爆日志。

**1～20 / S1：\(N_{\mathrm{enc}} \equiv 0\)**（代码硬门）。

### 7.2 单次遭遇结算（自动、瞬时）

不进入多轮回合。一次遭遇一步出结果：

1. 读地图海盗表，按权重抽 1 个海盗组（见 §7.4）。  
2. 计算  
   \[
   \mathrm{score} = \frac{\mathrm{Power}}{\mathrm{Power}+\mathrm{Enemy}} 
   \]
3. 掷骰得到：`大胜 / 险胜 / 战平(驱离) / 小败 / 大败`（阈值随 score 移动）。  
4. 应用结果表（§7.3），写一条简短战报。

展示：收杆图里增加「⚓ 护航战报」分区（最多显示 3～5 条摘要 +「另有 k 场从略」）。

### 7.3 结果表（设计意图）

| 结果 | 条件倾向 | 玩家感受 | 钓鱼层影响 | 海战层掉落 |
|------|----------|----------|------------|------------|
| 大胜 | score 高 | 猫很帅 | 无损失；小概率双倍本次片段鱼（慎用） | 高档海盗箱 |
| 险胜 | score 中高 | 惊险 | 无损失 | 普通箱 |
| 驱离 | 中等 | 平淡 | 无损失 | 极少材料 |
| 小败 | 偏低 | 被摸走罐头 | **损失本段 10%～20% 鱼获数量**（随机抽鱼） | 无 |
| 大败 | 很低 | 惨 | **损失 25%～40% 鱼获 + 少量饵** | 无；护卫猫大幅减轻 |

关键约束：

- **不允许**清空玩家全部背包，只动「本段结算鱼获缓冲」。  
- 护卫猫主要削失败档的损失比例，而不是无限提高胜率。  
- 炮手猫主要抬大胜/险胜权重与箱品质。  
- 军师猫主抬「完美护航」（整次收杆 0 失败）与主动海战收益。

### 7.4 海盗种类（草案图鉴）

| ID | 名称 | 大致 Enemy | 出现图 | 特色 |
|----|------|------------|--------|------|
| p01 | 吱吱扒手团 | 低 | 21～24 | 只偷小鱼 |
| p02 | 火药甲板鼠 | 中 | 23～27 | 失败时多扣饵 |
| p03 | 黑旗巡航队 | 中高 | 24～28 | 标准敌人 |
| p04 | 会计鼠精算班 | 特 | 26～29 | 专偷高价鱼（按价格权重抽） |
| p05 | 幽灵帆鼠 | 高 | 27～30 | 胜率 tail 更狠，掉幽灵材料 |
| p06 | 皇鼠亲卫 | 很高 | 29～30 | 精英，大胜掉稀有猫碎片 |
| p07 | 舵轮霸主（周常/主动） | BOSS | 30 或主动海战 | 日/周限次 |

### 7.5 主动 `海战`（可选日循环）

- 每日次数：**硬顶 3**（与 S2「日升级 3 次」同哲学：关键决策，不连点）。  
- 消耗：无 / 或 1 个「信号弹」材料。  
- 奖励：比自动遭遇更好的箱 + 更高整卡率。  
- 失败：不丢本段鱼获（主动战与挂机缓冲隔离），仅「猫掉一点点士气」或空奖，避免劝退。

## 8. 钓鱼层：21～30 如何接旧公式

### 8.1 延续

- 稀有度：仍用 `d = rod_level - location_difficulty` + 高斯裁剪（`概率和数值设计文档.md`）。  
- 价格：`price = base_n * C^r` 体系向 n=21…30 延拓（需重跑 `gen_table` / 回归 A、B，或对 21+ 单独校准）。  
- 鱼饵、鱼钩、收杆/状态结算流程复用。  
- UTR：建议 **对齐 11～20 星空图**（集齐 UR 后解锁 UTR + 迷途风可见），避免再发明第三套 UTR，降低维护成本。

### 8.2 刻意削弱/屏蔽的旧 buff（与 11～20 对称思考）

| Buff | 21～30 建议 |
|------|-------------|
| 玉米打窝 | ✓ 可用（或共享新上限，待定） |
| 展示木框 / 猫猫框 | **✗ 不生效**（同星空） |
| 星空艇 | **✗ 不生效** |
| S1 乐园建筑 buff | **✗ 不生效** |
| 雕像鱼竿 +1 | ✓ 全局仍生效（它是杆级本身） |
| **三猫 / 海战** | **✓ 仅 21～30** |

速度公式在 21～30 变为概念上：

```text
speed = (1+钩) * (1+(饵+打窝)/100) * 天气 * (海域专属乘区?)
```

**不要**把猫战力直接乘进出鱼速度——猫是「护航」，不是第二鱼钩，避免数值缠死。

### 8.3 海域专属小元素（轻量，可选 v1）

| 元素 | 说明 | 是否 v1 必做 |
|------|------|--------------|
| 海域天气 | 晴、雨、风暴、黑旗日、月夜 | 建议 v1 有 4～5 种 |
| 黑旗日 | 遭遇率↑、战利品↑ | 建议有 |
| 月夜 | 高稀有 soft 倾斜或军师收益↑ | 可 v1.1 |
| 潮汐 | 早晚潮改 threat 或速度 | 可砍 |

天气只作用于 21～30；旧图天气表不改。

## 9. 经济与掉落（边界）

### 9.1 货币

| 资源 | 是否鱼币 | 说明 |
|------|----------|------|
| 鱼币 | 是 | 卖鱼、商店；海盗箱可开少量鱼币 |
| 海域材料 A/B/C | 否 | 升猫、造船插件；**不可**在 1～20 使用 |
| 猫碎片 | 否 | 合成/升星 |
| 信号弹 | 否 | 主动海战（若启用） |

### 9.2 防崩约束

1. 海战期望鱼币 **≪** 同时长纯钓鱼期望（建议 ≤15%～25%），海战是「护航保险 + 轻度彩票」，不是第二主矿。  
2. 失败惩罚削的是本段鱼获，长期应使「不养猫硬闯高图」的净收益 **明显低于** 低图 + 养猫推进。  
3. 不引入可交易 P2P 猫卡（QQ 群游戏简化，防纠纷）；猫绑定账号。

### 9.3 成长目标（体验向，数值待模拟）

| 里程碑 | 目标体感 |
|--------|----------|
| D1 | 白给猫能在 21～22 图大概率不亏 |
| ~1 周 | 23～25 图稳定险胜+，开始攒升星 |
| ~2～3 周 | 26～28 图可挂机，主动海战打精英 |
| ~1 月 | 29～30 图成型；舵轮渊可周常 |

（可与 S1「约 20 天建设」、S2「约 30 天通关」错开压力，S3 是**长线主线渔场**而非短活动。）

## 14. 开放问题（需你拍板）

1. **入口门槛**：20 图成就推进 vs 单独「造海船」建设小游戏？  
   - 推荐：轻量「启航」（1～2 步材料），不要再做第二个 S1 规模建设。  
2. **猫成长维度**：等级 + 升星 是否都要？还是只等级 1～30 一条线？  
   - 推荐 v1：**仅等级 + 职业固定**，升星留 v1.1。  
3. **主动海战**：v1 做不做日 3 次？  
   - 推荐：**做**，反馈好且实现量可控。  
4. **失败惩罚手感**：丢本段鱼 10%～40% 是否过狠？  
   - 可用模拟决定；护卫猫成型后大败应罕见。  
5. **是否允许 21～30 使用新专属加速**（如「船帆」）？  
   - 推荐 v1 不做，只靠饵+钩+打窝+护航，系统更干净。  
6. **S3 是否需要独立「赛季结算」**？  
   - 推荐：主线永久渔场，不做限时赛季；若要活动可另开活动图。
