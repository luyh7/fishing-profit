# 6位幸运 ID 概率、番种与整数分分布

本版将号码空间改为 `000000` 到 `999999`，共 10^6 个 ID。规则沿用 9 位 v2 的最大番种设计，但全小/全大和星空飞机改为 6 位语境。

## 设计口径

1. 番种分值为 `-log10(番种出现概率)`。
2. 同家族取最大匹配，小番被大番包含时不重复计主分。
3. 同号吸收同号回文。
4. 全小/全大只在完整 6 位成立：全小为 0-4，全大为 5-9。
5. 6 位星空飞机：第 2 到第 5 位每一位都必须和左邻或右邻相同。
6. 对子：统计恰好长度为 2 的同号连段；两对 / 三对同家族，三对吸收两对。
7. 游戏展示分为原始总分四舍五入后的整数分。
