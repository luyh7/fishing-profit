# S2 · 猫猫挖矿（星穹矿脉）

> 状态：**v1.0 推翻重写** · 网页 Demo 起手  
> **完全独立**：不消耗鱼竿，不读写钓鱼金币；无天气/鱼饵/UTR/乐园。

## 一句话

外观像猫猫钓鱼的温和挂机，内核是 **挖矿三指令 + 稀有矿物 + 挖穿星球 + 大胆数值到 ∞（二阶无限通关）**。

## 定案约束

| 项 | 值 |
|----|----|
| 循环 | 挖矿 / 结束挖矿 / 挖矿状态 |
| 日升级 | **硬顶 3**（矿店） |
| 里程 | 深度挖穿星球 |
| ∞ | 一阶 `10^308`；通关需 **二阶无限 ≥ 1** |
| 结算 | 大批量 **概率分布** 一次出结果 |
| 目标节奏 | **~30 天** |
