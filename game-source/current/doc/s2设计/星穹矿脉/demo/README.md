# 猫猫挖矿 Demo · Web v1.0

外观对齐钓鱼温和挂机，**整套推翻**旧版指数破界模拟。

## 玩法摘要

- 指令循环：**挖矿 / 挖矿状态 / 结束挖矿**
- 矿店：钻头 / 镐速 / 矿车 / 爆破，**每日最多 3 次**
- 里程：深度挖穿星球
- 数值：可到一阶 `∞`（10^308），通关需 **二阶无限 ≥ 1** 且挖穿
- 结算：`n ≤ 40` 精确模拟；`n > 40` **多项分布正态近似**

## 启动

```powershell
# 推荐：钓鱼网页端隐秘路径（4159 已挂载）
# http://127.0.0.1:4159/_lab/xqkm/index.html

# 可选：单独静态服
python ".\web\serve.py"
# http://127.0.0.1:4160/
```
