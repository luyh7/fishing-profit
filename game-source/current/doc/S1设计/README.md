## S1 是什么

S1 对应钓鱼主线中的 **猫猫乐园** 特殊地图 / 常驻中型活动，与 1～20 图主线钓场相互独立入口，但共享钓鱼插件生态（鱼竿、鱼币、图鉴等）。
