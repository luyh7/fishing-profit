🌌 UTR星空剪影效果测试
测试鱼: {{ fish_name }} | 目标: 星空背景只在鱼轮廓内显示
原始鱼图
星空背景
方案A: CSS mask-image + file:// URL (当前方案)
background-image: 星空(file://) | mask-image: 鱼图(file://)
如果显示空白 → file:// URL不被mask-image支持
方案B: CSS mask-image + base64 Data URI
background-image: 星空(base64) | mask-image: 鱼图(base64)
如果显示正常 → 问题出在file:// URL; 仍空白 → mask-image本身不支持
方案C: Canvas合成 (globalCompositeOperation)
JS Canvas: 先绘制鱼图 → 设置source-in → 绘制星空 → toDataURL输出
纯JS Canvas合成，兼容性最佳
方案D: CSS mix-blend-mode ✅ 已采用
底层: 鱼图(filter:brightness(0)变黑剪影) | 上层: 星空(mix-blend-mode:screen) | 全部使用file:// URL，无需base64
screen模式: 黑色区域显示星空，浅色区域星空几乎不可见
方案E: SVG <mask> 元素
SVG mask: 用鱼图作为mask白色区域 → 星空rect填充被mask裁剪
SVG原生mask，用鱼图的亮度作为遮罩
方案F: CSS mask-image + base64 (80px背包尺寸)
与方案B相同，但使用背包页面的80px尺寸，验证尺寸是否影响
80px尺寸对照
